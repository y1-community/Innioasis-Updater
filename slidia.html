üçé macOS Installation

Copy and paste this command into Terminal and press Enter:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/team-slide/Innioasis-Updater/main/run_mac.sh)"

This will automatically set up everything you need!

After the first run, you can use the same command again to launch the app for future uses.

Note: This is a temporary preview release. An installer app is coming soon where you won't need the Terminal window.
